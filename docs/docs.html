Documentation.

Everything the engine understands, with a runnable example for each feature. Every output shown here is produced by the test suite.

Install and render

Create one engine per application, compile templates once, render as many times as you want. The engine is safe to share across goroutines.

main.go
go get github.com/victoragudo/go-veloz

engine := veloz.New()
tmpl, err := engine.Compile("hello", "Hola {{ name | capitalize }}!")
out, err := tmpl.Render(map[string]any{"name": "mireia"})
Hola Mireia!

Note compile errors are real errors: an unknown filter or a broken tag fails at Compile time, not in production at render time.

Expressions

Expressions work anywhere: inside {{ }}, in conditions, in filter arguments.

Literals

42, 3.5, 1.2e3, "text", 'text', true, false, null (alias none), arrays [1, 2, 3] and maps {city: "Madrid", "zip": "28001"}. Bare map keys become strings.

Arithmetic and strings

{{ 2 + 3 * 4 }}  {{ 10 / 4 }}  {{ 10.5 % 3 }}  {{ 2 ** 8 }}  {{ "go" ~ "-veloz" }}
14 2.5 1.5 256 go-veloz

Semantics integer math stays integer, division always returns a float, % works on floats too, ~ concatenates as strings, and == compares numbers exactly (true == 1 is true, large int64 values never lose precision).

Comparison, logic, membership

{{ age >= 18 and not banned }}
{{ 2 in [1, 2, 3] }}  {{ "a" in "cat" }}  {{ 9 not in [1, 2, 3] }}
true true true true

Short-circuit and / or return the deciding operand, so {{ user.name or "guest" }} prints the fallback when the name is empty. Truthiness: null, 0, "" and empty collections are false.

Ternary and elvis

{{ age >= 18 ? "adult" : "minor" }}   {{ nickname ?: "guest" }}
adult guest

Indexing

{{ items[0] }}  {{ items[-1] }}  {{ prices["basic"] }}  {{ matrix[1][0] }}

Negative indexes count from the end of the list.

Operator precedence

From lowest to highest: ternary and ?: · or · and · not · comparisons and in · ~ · + - · * / % · unary minus · ** · attribute, index, call and | filter. Note that a filter binds tighter than unary minus: write {{ (-4) | abs }}, not {{ -4 | abs }}.

Variables and data

Pass any Go value to Render. Maps, structs, slices and pointers all resolve naturally.

template + data
{{ user.email }}            map key
{{ product.Name }}          struct field
{{ product.Label() }}       method, no args
{{ product.Discounted(25) }} method with args
{{ order.lines[0].qty }}    nested access

Lenient by design a missing key or field renders as empty instead of failing. Methods can return (T) or (T, error). Struct access is cached by reflection type, so repeated renders do not pay reflection again.

For full control, implement the Attributer interface and skip reflection entirely:

func (u User) VelozAttr(name string) (veloz.Value, bool) {
    switch name {
    case "name": return veloz.Str(u.Name), true
    }
    return veloz.Nil(), false
}

If and set

{% if temperature > 30 %}hot{% elseif temperature > 15 %}warm{% else %}cold{% endif %}
warm

set creates or reassigns a local variable. Reassigning inside a loop is the idiomatic way to accumulate:

{% set total = 0 %}
{% for line in invoice.lines %}{% set total = total + line.total() %}{% endfor %}
Total: {{ total }}
Total: 1415

Loops

{% for name in names %}{{ name }}; {% endfor %}
{% for item, units in stock %}{{ item }}={{ units }} {% endfor %}
Mireia; Aleix; Nuria; keyboard=12 mouse=30

Maps iterate in sorted key order, so output is deterministic. On a slice, the two-variable form gives you the index: {% for i, name in names %}.

Empty collections: for / else

{% for x in results %}{{ x }}{% else %}No results found{% endfor %}

Inheritance and includes

A child template extends a layout and overrides its named blocks. Blocks keep their default content when not overridden. Both templates must be compiled in the same engine.

layout
<h1>{% block title %}Default{% endblock %}</h1>
<div>{% block body %}{% endblock %}</div>
<footer>{% block footer %}{% include "footer" %}{% endblock %}</footer>
page
{% extends "layout" %}
{% block title %}Profile of {{ name }}{% endblock %}
{% block body %}Content here{% endblock %}
<h1>Profile of Aleix</h1><div>Content here</div><footer>...</footer>

Rules chains of any depth work (page extends section extends base), the first definition of a block wins down the chain, inheritance cycles are detected and reported, and include renders another compiled template with the same data.

Autoescape

HTML escaping is on by default. Anything printed with {{ }} is escaped unless it is marked safe.

{{ html }}
{{ html | raw }}
&lt;b&gt;bold&lt;/b&gt; <b>bold</b>

Three escape hatches the raw filter, returning a veloz.SafeString from Go, or disabling it per engine with veloz.New(veloz.WithAutoescape(false)) when you generate non-HTML output like config files or code.

Scope escaping covers HTML body context (& < > " '). It is not context-aware like html/template: do not interpolate untrusted data inside href, onclick or <script>.

Whitespace control

Add a dash to any delimiter to eat the whitespace on that side: {{-, -}}, {%-, -%}.

a

  b
{%- endif -%}

Filters

Apply with | and chain freely: {{ names | reverse | join(", ") }}. Every filter can also be called as a function: upper("hola") equals "hola" | upper.

upperstrings

Converts to upper case.

{{ "hello" | upper }}HELLO
lowerstrings

Converts to lower case.

{{ "HELLO" | lower }}hello
capitalizestrings

Upper case first letter, lower case for the rest.

{{ "hello world" | capitalize }}Hello world
titlestrings

Upper case at the start of every word.

{{ "point of sale" | title }}Point Of Sale
trimstrings

Removes surrounding whitespace, or the given characters.

{{ "xxwordxx" | trim("x") }}word
replacestrings

Replaces every occurrence. Also accepts a map of pairs.

{{ "cat" | replace("c", "b") }}bat
splitstrings

Splits a string into a list by a separator.

{{ "a;b;c" | split(";") | join("+") }}a+b+c
lengthany · alias count

Items in a list or map, runes in a string.

{{ "azores" | length }}6
defaultany

Fallback value when the input is empty or false.

{{ "" | default("guest") }}guest
joinlists

Concatenates list items with a separator.

{{ names | join(", ") }}Mireia, Aleix, Nuria
reverselists · strings

Reverses a list or the runes of a string.

{{ "veloz" | reverse }}zolev
firstlists · strings

First element or first character.

{{ names | first }}Mireia
lastlists · strings

Last element or last character.

{{ names | last }}Nuria
keysmaps · lists

Sorted keys of a map, or the indexes of a list.

{{ stock | keys | join(",") }}keyboard,mouse
absnumbers

Absolute value, keeps ints as ints.

{{ (-7.5) | abs }}7.5
roundnumbers

Rounds to N decimals, default 0.

{{ 3.14159 | round(2) }}3.14
escapehtml · alias e

Escapes HTML and marks the result safe.

{{ "a & b" | escape }}a &amp; b
rawhtml

Marks a value as safe, skipping autoescape.

{{ html | raw }}<b>bold</b>
nl2brhtml

Escapes, then turns new lines into <br />.

{{ notes | nl2br }}first line<br />second line

Functions

Called with parentheses. Any function also works as a filter: {{ 5 | max(9) }} is max(5, 9).

FunctionDescriptionExampleOutput
range(start, end, step?)Inclusive numeric sequence, step defaults to 1 and can be negativerange(10, 0, -5) | join(",")10,5,0
max(...)Largest of the arguments or of a single listmax(3, 8, 2)8
min(...)Smallest of the arguments or of a single listmin([4, 1, 7])1
length(v)Same as the length filterlength("madrid")6

Custom filters and functions

A filter and a function share the same shape: a Callable that receives evaluated arguments and returns a value. For filters, the piped value arrives as args[0].

main.go
engine.RegisterFilter("money", func(args []veloz.Value) (veloz.Value, error) {
    f, _ := args[0].Interface().(float64)
    return veloz.Str(fmt.Sprintf("%.2f EUR", f)), nil
})

engine.RegisterFunction("today", func(args []veloz.Value) (veloz.Value, error) {
    return veloz.Str(time.Now().Format("2006-01-02")), nil
})
{{ price | money }} → 49.90 EUR {{ today() }} → 2026-07-07

Value helpers build return values with veloz.Str, veloz.Int, veloz.Float, veloz.Bool, veloz.Nil, veloz.Object or veloz.FromAny. Read arguments with .Interface(), .String(), .IsTruthy(). Return a veloz.SafeString wrapped in Object to bypass autoescape.

Go API

CallWhat it does
veloz.New(opts...)Creates an engine with the built-in filters and functions registered
veloz.WithAutoescape(false)Option: disable HTML escaping for this engine
engine.Compile(name, src)Parses and compiles to bytecode, registers the template under its name
engine.MustCompile(name, src)Same, panics on error. For templates known at build time
engine.Get(name)Returns an already compiled template
tmpl.Render(data)Renders to a string
tmpl.RenderTo(w, data)Renders straight into an io.Writer, no intermediate copy
tmpl.RenderBytes(data)Renders to a fresh byte slice
engine.RegisterFilter(name, fn)Adds a filter, available to every template of the engine
engine.RegisterFunction(name, fn)Adds a function, same namespace as filters

Concurrency one engine serves all goroutines. Interpreters, stacks and locals live in an internal pool, so steady-state rendering does not allocate fixed overhead.
